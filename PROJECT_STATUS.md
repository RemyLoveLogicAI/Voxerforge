## 🚀 How to Use

### Quick Start
```bash
git clone https://github.com/RemyLoveLogicAI/Voxerforge.git
cd Voxerforge
./setup.sh
npm run web:dev
```

### Build Everything
```bash
npm install
npm run build
```

### Individual Packages
```bash
cd packages/core && npm run build
cd packages/voice && npm run build
cd packages/ai && npm run build
cd apps/web && npm run build
```
